Geração de Arquivos PDF utilizando pdfkit

Com o pdfkit é possível gerar arquivos PDFs a partir de código HTML. Também é possível estilizar o arquivo utilizando CSS!

Com pdfkit é possível gerar arquivos PDF a partir de:

  • De sites, a partir de URLs;
  • De arquivos HTML (como esse exemplo);
  • Diretamente de código HTML em strings.

Podemos incluir imagens, bastando convertê-las para base64!
